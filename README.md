## dataBase_dataget
オラクルのデータベースからデータを取り込みデータフレーム化を実施するコード

## プログラムの概要
このスクリプトは、以下のステップで動作します。
接続情報の設定: データベースの場所や認証情報を定義。
DSNの作成: Oracle特有の接続文字列（Data Source Name）を生成。
接続と実行: データベースに接続し、SQLを実行してデータを取り込む。
終了処理: 処理の成否にかかわらず、確実に接続をクローズする。

## 接続設定（パラメータ）
HOST / PORT / SVS: データベースサーバーの住所（IPやホスト名）、ポート番号、サービス名。
USER / PASS: データベースにログインするためのユーザー名とパスワード。
sql_query: 実行するSQL文。ここでは「指定したテーブルから先頭300件を取得する」内容になっています。

## メイン処理（try-except-finally）
エラー発生を防ぎ、安全にデータベースを操作するための構造になっています。
dsn = cx_Oracle.makedsn(...)
ホスト、ポート、サービス名を合体させて、Oracle専用の接続先情報を作成します。
conn = cx_Oracle.connect(...)
実際にデータベースへのログインを試みます。
df = pd.read_sql(sql_query, conn)
重要ポイント: SQLの実行と結果のDataFrame変換を同時に行います。
except Exception as e
パスワード間違いやネットワークエラーなど、何か問題が起きた時にエラーメッセージを表示し、プログラムが強制終了するのを防ぎます。
finally
重要ポイント: 成功しても失敗しても、最後に必ず conn.close() を実行し、データベースの接続枠（セッション）を解放します。

## 注意点と補足
セキュリティ:
コード内にパスワードが直接記述されています。
本番環境や共有環境では、環境変数や設定ファイルから読み込むように変更することが推奨されます。
環境依存:
cx_Oracle を動かすには、実行環境（PCやサーバー）に Oracle Instant Client がインストールされている必要があります。
